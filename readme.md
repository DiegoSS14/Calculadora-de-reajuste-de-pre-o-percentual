# Calculadora de Reajuste de Preço

Exercício básico de PHP que implementa uma calculadora para aplicar um ajuste de preço por meio de porcentagem informada pelo usuário.

## Objetivo

Aplicar conhecimentos fundamentais de:
- **HTML** - Estrutura do formulário
- **CSS** - Estilização da interface
- **PHP** - Lógica de processamento dos dados

## Executando o Servidor

Abra o terminal na pasta do projeto e execute:

```bash
php -S localhost:8000
```

Depois acesse `http://localhost:8000` no seu navegador.

## Como Usar

1. Acesse `http://localhost:8000/index.php` no navegador
2. Informe o preço atual
3. Informe a porcentagem de ajuste
4. Clique no botão para calcular
5. O resultado mostrará o novo preço com o reajuste aplicado
